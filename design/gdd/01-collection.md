# 01 · 收藏图鉴与养成 GDD（Collection Codex & Cultivation）

> 机制编号：M1　|　支柱：P2 收藏即求知　|　动词：收集（Collect）
> 版本：v1.0　|　作者：文策渊

---

## 1. 概述（Overview）

把既有的"收藏布尔"升级为**图鉴所有权系统 + 双轨养成**。玩家通过对话/召唤收集 59 位哲学家，解锁 35 流派与 3 时代图鉴，并用 `思想力(TP)` 培养单卡等级 `CL`，用全局 `学者等级 SL` 解锁四机制的功能门。

**与既有产品衔接**：直接复用 `EnhancedPhilosopherCard.onFavorite(id)` 事件；首次收藏写入 `collections` 表并触发奖励。哲学家实体只读，来自 `philosophers.ts`。

---

## 2. 设计目标（Design Goals）

| 目标 | 量化 |
|---|---|
| G1 降低收集门槛 | 对话即收藏，0 货币成本（D2：7 日人均 ≥8） |
| G2 提供可见成长 | SL/CL 双轨，每日可感知进度（SDT 胜任） |
| G3 驱动知识探索 | 流派/时代图鉴里程碑奖励（探索者） |
| G4 不通胀 | TP 有日上限与明确消耗（D7：周增速 ≤15%） |

---

## 3. 核心循环（Core Loop）

```
每日登录 → 签到(+50 TP / 活跃度)
   → 与未收藏哲学家对话 → 点收藏(首次 +120 TP, 写入图鉴)
   → 消耗 TP 升 CL(增强该卡辩论/阵容属性)
   → 达成图鉴里程碑(流派全收集/时代全收集) → 领奖励
   → SL 经验累积 → 解锁新机制门(辩论/阵容/实验)
   ↺ 次日
```

**单次会话微循环**：打开卡片 → 对话 → 收藏 → 看图鉴进度 → 升 CL。

---

## 4. 规则与状态（Rules & States）

### 4.1 图鉴所有权
- 状态：`uncollected`（未触达）/ `collected`（已收藏，永久）/ `cultivated`（CL≥5）。
- 首次收藏：触发 `onFirstCollect(philosopherId)` → 写 `collections` 行 + 发 TP+120 + SL 经验 +50。
- 重复点击收藏：仅切换 `isFavorited` 展示态，不发奖励。

### 4.2 流派图鉴
- 解锁条件：某流派 `school` 下**已收藏数 / 该流派总数 ≥ 60%** → 解锁"流派页"（展示该流派全部哲学家、关系、主题）。
- 完成条件：该流派 100% 收藏 → 发 CR+3、TP+500、SL 经验 +300。

### 4.3 时代图鉴（era）
- ancient / modern / contemporary 三时代，各 100% 收藏 → 发 CR+5、TP+800、SL 经验 +500、专属称号。

### 4.4 卡牌等级 CL（1→20，突破→30）
- 升级消耗：见 balancing.md §2（`E_CL(L)=round(50·L^1.15)`）。
- CL 效果：辩论维度加成 +1.2%/级（封顶 +24% @CL20）；阵容基础属性 `ATK=100+CL·3`。
- 突破：CL20→25 耗 CR10；25→30 耗 CR20（每哲学家独立）。

### 4.5 学者等级 SL（1→60）
- 升级消耗：见 balancing.md §2（`E_SL(L)=round(150·L^1.25)`）。
- 功能门（gating）：

| SL | 解锁 |
|---|---|
| 1 | 图鉴 |
| 3 | 灵感召唤 |
| 5 | PVE 辩论 |
| 8 | 阵容（4 槽） |
| 10 | 思想实验·第一章 |
| 15 | PVP |
| 20 | 阵容槽 5 前置（耗 RP） |
| 30 | 卡牌觉醒 |
| 40 | 思想实验·终章 |
| 60 | 满级称号"集哲者" |

### 4.6 成就系统（Should）
- 类别：收集类（收藏 N 位 / 全流派 / 全时代）、养成类（单卡 CL20 / 总 CL 和≥200）、社交类（首胜 PVP）、探索类（走完某主题全分支）。
- 奖励：TP/IN/CR + 称号，写入 `achievements` 表。

## 5. 数值与平衡（Numbers & Balance）

> 全部数值以 `balancing.md` 为权威，此处仅给关键值与示例。

**示例 1（CL 升级）**：将苏格拉底从 CL5 升到 CL6，需 `E_CL(5)=round(50·5^1.15)=round(50·6.22)=311 TP`。
**示例 2（流派里程碑）**：儒家约含 8 位（孔子/孟子/荀子/董仲舒/朱熹/王阳明/…），收藏满 8 位 → 领 CR+3、TP+500。
**示例 3（日获取上限）**：免费行为 TP ≤800/日（签到 50 + 新收藏 120×6 封顶 720）；对战 TP ≤1200/日；合计 ≤2000/日。
**示例 4（SL 进度）**：从 SL1 到 SL5 累计经验 ≈ 150+357+592+849+1121 = 3069，普通玩家约 5–7 日可达（解锁 PVE 辩论）。

**防通胀**：TP 不可交易、无产出放大器；IN/CR 有仓库上限（9999 / 999）；周常 CR 限 10/周。

## 6. 反馈与表现（Feedback & Presentation）

| 事件 | 表现 |
|---|---|
| 首次收藏 | 卡片金光扫过 + "收录图鉴"弹层 + TP+120 飘字 |
| CL 升级 | 卡面等级数字滚动 + 属性条增长动画 |
| 流派/时代完成 | 图鉴页全亮 + 专属印章 + 成就 toast |
| SL 升级 | 顶部进度环突破 + 功能门解锁提示 |
| 每日签到 | 7 日日历翻牌，第 7 日额外 IN+50 |

**UX 落点**：图鉴页作为新 Tab，沿用既有 FilterBar 的流派/主题/年代筛选；卡片右下角展示 CL 徽章与收藏态，不增加主网格视觉负荷（红线 3）。

---

## 7. 风险与缓解（Risks & Mitigation）

| 风险 | 缓解 |
|---|---|
| R1 收集后期无目标（59 位收满后空虚） | 转向 CL 养成、流派/时代精 completion、觉醒（Could） |
| R2 TP 囤积不用 | 限时"思想力涌动"活动双倍消耗返利；CL 突破消耗 CR 制造稀缺 |
| R3 新手不知先养谁 | 新手 7 日引导任务指定 3 位低门槛哲学家（苏格拉底/庄子/边沁） |
| R4 后端表缺失导致进度丢失 | `collections/progress` 表 + 客户端 localStorage 兜底同步 |

---

## 8. 验收标准（Acceptance Criteria）

- [ ] `onFavorite` 首次触发写入 `collections` 且发 TP+120，重复点击不发。
- [ ] 图鉴页展示 59 位收藏态、35 流派完成度、3 时代完成度。
- [ ] CL 升级按 `E_CL` 扣 TP，ATK/辩论加成实时生效。
- [ ] SL 达门槛自动解锁对应机制门并提示。
- [ ] 流派/时代 100% 触发里程碑奖励且幂等（不重复发）。
- [ ] 日 TP 获取不超 2000，仓库 IN≤9999 / CR≤999 硬截断。
- [ ] 指标 D2（7 日人均收藏 ≥8）灰度达标。

> 跨系统资源与曲线见 `balancing.md`；冲突仲裁见 `consistency-review.md`。
